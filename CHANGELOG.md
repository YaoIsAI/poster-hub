## [1.0.0] - 2026-04-07

### Added
- 初始版本发布
- GitHub URL 自动解析（stars、语言、描述）
- 本地项目路径支持
- 纯文字描述生成海报
- 多主题支持：Apple 极简 / 深色科技 / 赛博朋克 / Python 官方 / 库存管理
- 中英双语界面和海报
- Playwright 高清截图输出
- Web 界面（生成器 + 画廊）
- REST API（/api/generate, /api/poster/:id.png, /api/list）
- OpenClaw AI 助手技能集成（SKILL.md）
- 命令行工具
- 本地 LLM 分析支持
